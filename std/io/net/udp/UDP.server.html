    Function UDP.server

    Create a UDP socket and bind it to the resolved hostname and port. Uses the first resolved address the socket can successfully bind. A port number of zero will be bound to an ephemeral port.

    UDP server(S) (
      S hostname,
      ushort port = 0
    )
    if (isStringLike!S);

    Parameters

    NameDescription
    hostname hostname to resolve and bind
    port port number to bind

    Example

    auto server = UDP.server("localhost", 1234);
auto client = UDP(server.localAddr.family);
ubyte[4] ping = ['p', 'i', 'n', 'g'];
client.sendTo(server.localAddr, ping[]);
ubyte[4] buf;
assert(server.recv(buf[]) == 4);
assert(buf[] == ping[]);
